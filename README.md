# 💻 Sistema de Caixa Eletrônico em Python

## 📌 Descrição

Este projeto simula um caixa eletrônico (ATM) desenvolvido em Python, utilizando os principais conceitos de Programação Orientada a Objetos (POO).

## 🚀 Funcionalidades

* Criar conta (corrente ou poupança)
* Depositar dinheiro
* Sacar dinheiro
* Consultar saldo
* Visualizar histórico de operações

## 🧩 Conceitos de POO aplicados

* Encapsulamento
* Herança
* Polimorfismo
* Agregação
* Composição

## 🛠️ Tecnologias

* Python

## ▶️ Como executar

1. Clone o repositório:

```
git clone https://github.com/YanSantos1/Avalia-o-P.O
```

2. Acesse a pasta do projeto:

```
cd nome-do-repositorio/banco
```

3. Execute o sistema:

```
python main.py
```
